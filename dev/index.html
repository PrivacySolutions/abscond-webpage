The Abscond bundle, the gateway to the darknets

A web without tracking and mass surveillance.

Decentralized messaging system

After downloading the Abscond bundle you will have access to a system that can transfer messages (like email) between clients. You don't need to rely on a mail provider like TorMail or Lavabit, which both have been closed down by some kind of government or others.

No one tracks you!

I2P and Tor are designed to make it hard, or even sometimes impossible for analytics to track you. This is due to the actual designs of those darknets, which are made this way to ensure your privacy. The browser also ships with anti tracker plugins.

Built-in Torrent client

The I2P router ships with a built-in Torrent client which runs in the background. This client is provides a simple webui with some configuration options. This tool can be handy to share big files.

All traffic is encrypted

All incoming and outgoing traffic from the Abscond bundle except for the reseed part that will fallback to http if https is unavailable. The encryption is done in multiple levels increasing your privacy. Please read more on mix-networks to get a better understanding of how I2P and Tor works.

Features

Browser plugins

The Abscond bundle ships with NoScript, DoNotTrackMe, AdBlock, Wappalyzer, Webdeveloper and some other snacks.

I2P & Java (JRE)

The I2P router is the most important feature in this package. It does all the routing, as well as forwarding onion and clearnet requests. JRE is included.

Free software/Open Source

The Abscond bundle has only one closed source compnent, Java. All other components are open source, and free for all to download.

Firefox forked browser

Since this is a fork of Firefox, you will have the same compability against websites as with a normal Firefox, except the stronger privacy policies. This also allows you to install Firefox themes and extensions.

Talk with people without listeners

These days it's hard to get under the radar, almost everything you write on the public internet will be monitored by some kind of group or government. I2P and other similar networks might be the only place left on the internet where you can't easily get wiretapped.

Highly configurable

Both the Abscond browser and I2P itself is highly configurable, you got settings on low levels that decide the internal workings of the application, and up to themes and colors options. However it's recommended to get known with the system before start customizing. Remeber customizing could leak your identity if you do something wrong.

Portable

The Abscond bundle also handles to be in a dynamic environment, making it possible to for example have a USB stick containing your bundle, which you use on random computers you bypass.

Universal software

The Abscond bundle can currently run on Windows, Linux and Mac OS X. Probably BSD/Solaris and a few others too, but that isn't tested. Support for Android will be added.

Currently the Firefox fork was forked at version 24.6.0, I2P is at version 0.9.13 and OpenJDK is 1.7.

Question #6: Why does the browser block javascript per default? And why don't darknet people like javascript?

Javascript can do a lot, and it's code hosted on a server that can execute inside your browser. If this code is a form of malware it can deanonymize you, listen on your keyboard when using the internet, detect other hosts on your LAN. And much more. That's why few sites have any javascript at all, because they don't expect the users to have it enabled becasue of security reasons. NoScript gives you a whitelist, but try to keep it a minimal for your own safety.

The Abscond bundle

The Abscond bundle was built for giving the average user a easy access to the darknet I2P. The bundle includes a browser, torrent client, a anonymous decentralized messaging client, java and I2P itself. You might have tried the Tor Browser bundle before, this is kind of the same just for I2P and Tor combinded. But our main focus is the I2P part of it. Tor access is enabled via an I2P plugin.
