The Abscond Browser Bundle

your gateway to the darknets

Decentralized messaging system

The Abscond Browser Bundle allows you to transfer messages between clients (like email). You won't need to rely on an email provider like TorMail or Lavabit (which both have been closed down by some kind of government or others.)

No one tracks you!

I2P and Tor are designed to make it hard, or even impossible for analytics to track you. This is due to the actual designs of both networks which are made this way to ensure your privacy. The browser makes use of these designs and also ships with anti-tracker plugins.

Built-in Torrent client

The I2P router ships with a built-in anonymous torrent client which provides a simple webui and some configuration options. This tool can be handy to share any file, big or small - anonymously!

All traffic is encrypted

All incoming and outgoing traffic is encrypted (except for some essential I2P internals, but you will still remain anonymous). The encryption is done in multiple levels thus increasing your privacy.

Why use the Abscond Browser Bundle?

Features

Browser plugins

The Abscond Browser Bundle ships with NoScript, DoNotTrackMe, AdBlock, Wappalyzer, Webdeveloper and some other tools.

I2P & Java (JRE)

The I2P router does all the encrypted routing, as well as forwarding onion and clearnet requests. JRE is included with the Bundle.

Free software/Open Source

The Abscond Browser Bundle has only one closed source component: Java, but all other components are open source, and free for all to download.

Firefox forked browser

As this is a fork of Firefox, it works with clearnet websites like Firefox, except the stronger privacy policies. You can also install Firefox themes and extensions.

Talk without eavesdropping

Nearly everything you do on the public internet will be monitored by some group or government. I2P and other similar networks may be the only place left on the internet where you can't easily be wiretapped.

Highly configurable

Both the Abscond browser and I2P itself is highly configurable. It's recommended to become familiar with the system before you customize. Remeber: customizing could leak your identity if you do something wrong.

Portable

The Abscond Browser Bundle is dynamic, making it possible to have a USB key containing the bundle which you could then use on any computer of your chosing.

Universal software

The Abscond Browser Bundle currently runs on Windows, Linux and Mac OS X. BSD/Solaris and a few others OS's may work too, but they haven't been tested. Support for Android will be added.

Currently the Firefox fork was forked at version 24.6.0, I2P is at version 0.9.13 and OpenJDK is 1.7.

Question #6: Why does the browser block javascript per default? And why don't darknet people like javascript?

Javascript can do a lot, and it's code hosted on a server that can execute inside your browser. If this code is a form of malware it can deanonymize you, listen on your keyboard when using the internet, detect other hosts on your LAN. And much more. That's why few sites have any javascript at all, because they don't expect the users to have it enabled becasue of security reasons. NoScript gives you a whitelist, but try to keep it a minimal for your own safety.

The Abscond Browser Bundle

The Abscond Browser Bundle was built for giving the average user easy access to I2P and Tor. The bundle includes a browser, torrent client, a anonymous decentralized messaging client, java, and I2P itself. If you have used the Tor Browser Bundle before, this is somewhat the same but for both I2P and Tor COMBINED! Our main focus enabling I2P access but Tor is enabled via an I2P plugin.
